# Frontend (Story 9)

React + TypeScript + Vite shell for OpenClaw Monitor UI.

## Setup
```bash
cd frontend
npm install
```

## Run Locally
```bash
cd frontend
npm run dev
```

By default, API requests target the same origin. To point at a backend URL, set:

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000
```

## Build
```bash
cd frontend
npm run build
```

## Test
```bash
cd frontend
npm test
```
